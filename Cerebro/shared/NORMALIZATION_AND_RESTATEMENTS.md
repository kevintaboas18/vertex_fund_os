# Normalization and Restatements

## Normalized operating results

Create a reconciliation from reported to normalized values. Potential adjustments include:

- discontinued operations;
- restructuring that is genuinely non-recurring;
- acquisition expenses and integration charges;
- litigation settlements;
- unusual gains/losses;
- mark-to-market items unrelated to operating performance;
- stock-based compensation, shown both included and excluded but never silently removed;
- capitalized R&D for valuation analysis;
- operating leases treated as debt where the selected model requires it.

## Rule against aggressive normalization

An item recurring in three of five years is not automatically non-recurring. Keep it in the base case unless there is specific evidence that it will cease.

## Restatements

- Use the latest restated historical series.
- Preserve the original values in the audit appendix.
- Recalculate every dependent metric.
- In backtests, use only the values known at that historical timestamp to avoid look-ahead bias.

## Share count

Use fully diluted shares for per-share value. Include options, restricted units, convertibles, and contingent consideration under the selected treasury-stock or if-converted method.
